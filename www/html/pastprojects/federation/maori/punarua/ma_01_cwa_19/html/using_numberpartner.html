Using The Number Partner

The Number Partner shows you how to break up numbers into pairs. The Number Partner uses numbers greater than 10 and less than 30.

Once you have chosen a number, The Number Partner will it break up into three different number pairs. You will need to work out a sum to complete each number pair.

  1. Selecting Hint will display some strategies to help you break up the number and complete the number pair.
  2. Selecting New number will take you back to the beginning where you can start again with a different number.
  3. Selecting Help will display a help screen with instructions on how to use The Number Partner.
  4. Move this slider left or right to break your numbers into two parts.
  5. This is the answer box where you enter a number to complete the number pair. You can enter your answer at any time.
  6. Selecting Show ones displays the ones in the number bar. If you get stuck, you can use the ones to count-on to complete the number pair.
  7. Selecting Show pairs will show you a list of the number pairs you have completed.
