Control instructions

You can use The Number Partner with the keyboard or mouse.

Mouse instructions:
Move around the screen and select a button or select and move the sliders.
Keyboard instructions:
Use the Tab key to move to the different elements of the screen.
Use the Shift + Tab keys to move backwards through the screen.
Press the Enter or Return key or the Space bar to use the button that you used the Tab key to select.
Use the arrow keys to move the sliders.
